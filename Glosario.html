💻TIPOS DE ETIQUETAS💻

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

ESTRUCTURA DE ETIQUETAS:
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales como texto img entre otros.

🔎Listado de etiquetas dirigidas a objetos:

  • <img> sierve para agregar imagenes.
  • <header></header> contiene al encabezado de una sección o documento.
  • <section></section> se usa para representar una parte de un documento o artículo. Una regla general para determinar si su uso es correcto, es numerarlos.
  • <a></a> Es una etiqueta que nos ayuda a poder crear un enlace a una página web.
  • <footer></footer> El elemento footer representa al pie de una sección o documento, donde los autores habitualmente colocan firmas.
  • <table></table> sería el elemento que contendría todos los elementos de la tabla
  • <button></button> Es el elemento crea botones marcadores.
  • <article></article>representa una composición auto-contenida en un documento

🔎Listado de propiedades css dirigidas a texto:

  • color sirve para poner color al texto.
  • font-size Especifica la dimensión de la letra. Este tamaño puede, a su vez, alterar el aspecto de alguna otra cosa.
  • font-family Define una lista de fuentes o familias de fuentes, con un orden de prioridad, para utilizar en un elemento seleccionado.
  • font-weight Permite establecer el grosor con el que se muestran las letras del texto.
  • font-variant Es una de las menos utilizadas de CSS porque sólo permite seleccionar una variante.
  • font-strech Es el tamaño de letra del elemento contenedor.
  • font-kerning Es la posibilidad de acercar más o menos dos letras consecutivas aprovechando la forma de las letras.
  • border sirve para poner un borde y establecer el grosor.

🔎Listado de propiedades css dirigidas a objetos:

  • width Establece el ancho de un elemento mediante un valor numérico.
  • margin Establecer uno, dos, tres o los cuatro márgenes de forma simultánea.
  • border Se utiliza para establecer el mismo grosor list: nos permite crear conjuntos de elementos en forma de lista dentro de una página.
  • list Nos permite crear conjuntos de elementos en forma de lista dentro de una página
